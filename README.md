# Relebohile-Lethoko
This project is a five page podcast website designed to showcase radio show called Campus Voices Podcast. The website allows users to explore episodes, listen to audio content, learn about the hosts, and get in touch.
PAGES INCLUDED (index.html)
Introduces the podcast and features the latest episode.
EPISODE PAGE (index.html)
Displays a list of available podcast episodes.
EPISODE DETAIL PAGE (episode1.html)
Contains detailed information about episodes, including:
. Embedded audio player
. Interview content
. Show notes
HOSTS PAGE (hosts.html)
Provides profiles of the podcast hosts with images and biographies.
CONTACT PAGE (contact.html)
Includes a contact form for users to send messages.
